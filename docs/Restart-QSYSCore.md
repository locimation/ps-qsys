# Restart-QSYSCore

## SYNOPSIS
Restarts the Q-SYS core.

## SYNTAX

```
Restart-QSYSCore [-Session] <QSYS_Session> [<CommonParameters>]
```

## DESCRIPTION
Restarts the Q-SYS core.

## EXAMPLES

### EXAMPLE 1
```
Restart-QSYSCore -Session $qs
```
